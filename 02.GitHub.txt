
建立這台電腦的使用者名稱
    git config --global user.name "hcy"

查詢這台電腦的使用者名稱
    git config user.name
    
建立電子信箱
    git config --global user.email "a0910308890@gmail.com"

在專案目錄 啟用git
    git init

查看目前檔案狀態
    git status

將檔案建立追蹤
    git add .

儲存log
    git commit -m "v20191007#1"

推送log
    git push -u origin master

將線上最新的版本更新下來
    git pull --rebase
